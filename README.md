This repo is provides 3 step commands in implementing a feature. 
1. Plan
2. Review
3. Implement

Place this ai-flows/ repo in the same set of repos you are working on.
Lets say you have 3 repos 

fire-and-ash/
  fire-and-ash-frontend/
  fire-and-ash-backend/
  fire-and-ash-admin/

Now place the ai-flows here:

fire-and-ash/
  fire-and-ash-frontend/
  fire-and-ash-backend/
  fire-and-ash-admin/
  ai-flows/
